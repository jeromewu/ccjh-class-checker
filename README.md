# CCJH Class Checker

Mobile app to check CCJH class.

## Environment

As the environment of this app is pretty old, making sure you are using:

- Nodejs v14
- Expo-CLI 4.1.3
- Expo v2.14.0 (adb install archive/Expo_v2.14.0.apk)
- Android 7

## Setup

- Install dependencies

```bash
npm install
```

- Start development

```bash
npm run android
```
